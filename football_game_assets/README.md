# 1v1 Football Game

A simple 2D football game built with Pygame, featuring players like Messi, Neymar, and Ronaldo. Play 1v1 with a friend, move your player, kick the ball, and score goals!

## Requirements
- Python 3.6 or later
- Pygame (pip install pygame)

## Installation
1. Ensure Python is installed on your system.
2. Install Pygame by running:
   ```
   pip install pygame
   ```

3. Copy the football.py file to your desired directory.

## How to Run
1. Navigate to the directory containing football.py.
2. Run the game:
   ```
   python football.py
   ```

## Controls
- Player 1 (Messi):
  - Move: WASD keys
  - Kick: Spacebar
- Player 2 (Ronaldo):
  - Move: Arrow keys
  - Kick: Enter/Return key

## Gameplay
- Move your player to the ball and press the kick button to shoot.
- Score goals by getting the ball into the opponent’s goal (left or right side).
- The score is displayed at the top (Player 1 - Player 2).
- After a goal, players and the ball reset to starting positions.
- Close the window to quit the game.

## Player Stats
- Messi: Speed: 7, Shoot: 8
- Neymar: Speed: 8, Shoot: 7
- Ronaldo: Speed: 6, Shoot: 9

## Notes
- This version defaults to Messi vs. Ronaldo. To choose other players (e.g., Neymar), the code needs modification.
- The game uses basic graphics (circles for players, no sprites). For custom images, additional setup is required.
- For issues or enhancements, refer to the code comments or seek assistance.

Enjoy the game!
